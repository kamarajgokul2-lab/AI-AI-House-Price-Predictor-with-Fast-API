# Estima.AI — AI House Price Predictor

Estima.AI is a full-stack demo application that predicts house prices using a machine learning model served via FastAPI, paired with a modern, responsive single-page frontend.

![Status](https://img.shields.io/badge/status-demo-orange) ![License](https://img.shields.io/badge/license-MIT-blue)

## Features

- **AI-powered predictions** — sends property details to a FastAPI `/predict` endpoint and returns an instant price estimate
- **Modern UI** — architectural-blueprint inspired design system with dark/light mode
- **Fully responsive** — works across mobile, tablet, and desktop
- **Prediction dashboard** — total predictions, average, highest, and lowest values
- **History tracking** — local prediction history with search and category filters
- **Interactive charts** — prediction trend and area-vs-price visualizations
- **Export & share** — download a text report, copy results, or share via the native share sheet
- **Offline fallback** — if the API is unreachable, a local simulation provides an estimate so the UI never breaks

## Getting Started

### 1. Backend (FastAPI)

```bash
# Create a virtual environment (recommended)
python -m venv venv
source venv/bin/activate   # On Windows: venv\Scripts\activate

# Install dependencies
pip install -r requirements.txt

# Run the API
uvicorn main:app --reload --port 8000
```

The API will be available at `http://localhost:8000`.

**Endpoint:**

```
POST /predict
Content-Type: application/json

Request:
{
  "area": 1200
}

Response:
{
  "area": 1200,
  "predicted_price": 2450000
}
```

### 2. Frontend

No build step required — just open `index.html` in a browser, or serve it:

```bash
python -m http.server 5500
```

Then visit `http://localhost:5500`.

> By default, the frontend points to `http://localhost:8000` for API calls. Update the `API_BASE_URL` constant in `index.html` to match your backend's address if it differs.

## Customizing the Model

`main.py` currently uses a simple linear formula (`base price + area × price-per-sqft`). To plug in a real trained model:

1. Train and export your model (e.g. with `scikit-learn` + `joblib`).
2. Load it in `main.py`:
   ```python
   import joblib
   model = joblib.load("model.pkl")
   ```
3. Replace the prediction logic inside the `/predict` route with `model.predict(...)`.
